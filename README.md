```shell

├── README.md
├── doc
│   ├── assets
│   ├── design.md
│   ├── proposal.md
│   └── xxx_lib.md
├── src
│   └── Template.cj

    ├── LLT
    └── UT
```

- `doc`是库的设计文档、提案、库的使用文档
- `src`是库源码目录
